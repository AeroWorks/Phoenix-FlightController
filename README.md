Flight Controller implementation using MK20DX128 (Teensy 3.0)
=============================================================

PIN setup (Teensy 3.0 pin numbering)
------------------------------------

- I2C SCL - 19
- I2C SDA - 18

- PPM in - 3

- Rotor 1 - 22
- Rotor 2 - 23
- Rotor 3 - 9
- Rotor 4 - 10

- Orientation lights - 14
- Battery monitor - 15
